Ubuntu 8.04: Totem mit Youtube Plugin

2008-03-26, 06:31
Beta / Hardy / Totem / Ubuntu / Youtube

Ubuntu 8.04 Hardy Heron kommt mit einem frischen Totem Videoplayer mit Youtube Plugin welches allerdings in der aktuellen Hardy Beta noch etwas angeschubst werden muss. Das Plugin lässt sich bei einigen Installationen nicht so einfach aktivieren und bricht mit der (nicht sehr aussagekräftigen) Fehlermeldung: Plugin Fehler “Plugin »YouTube-Browser« konnte nicht aktiviert werden” ab. Abhilfe schafft die Installation des Pakets python-gdata welches für das Plugin benötigt wird, dieses kann man mittels: sudo apt-get install python-gdata aus den Paketquellen installieren.
